# Recommend_Book
Something to help me discover new books! 


##### To Use: 
1. Navigate to project 
    - cd ~/Desktop/Recommend_Book
2. Set up environment 
    - source venv/bin/activate
3. Run project 
    - python run.py "Book Title Here"
